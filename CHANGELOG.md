## [Unreleased]

### Added

- Hybrid static/dynamic epilepsy scanner with comprehensive tests.
- New CLI for scanning files and directories with options for dynamic rendering and quality scoring.
- Dynamic rendering module with a mock renderer and flash detection algorithm.
- Safety registry for risk events and integrated it into the static analysis.
- Quality analysis module to assess background types based on shader assignments.
- Extensive unit tests for lexer, parser, analyzer, and scanner functionalities.
- Reporting system to generate EARL-compliant reports for scan results.
- Constants for tool identification and PES schema URI.
